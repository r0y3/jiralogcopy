# JIRA Log Copy

## Get Started

1. Clone this repo

    ```
    $ git clone https://github.com/royge/jiralogcopy.git
    $ cd jiralogcopy
    ```

1. Install requirements

    ```
    $ pip install -r requirements.txt
    ```

1. Modify `settings.py`

1. Make `main.py` executable

    ```
    $ sudo chmod +x main.py
    ```

1. Run `main.py`

    ```
    $ ./main.py
    ```
